# Visually AI

A simple but really useful browser extension for frontend devs. Visually edit any webpage and generate GitHub Copilot CLI commands from your changes.

## Installation

1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Turn on **Developer Mode** (top right toggle).
4. Click **Load unpacked**.
5. Select the `visually-main` folder.
6. Done! Click the Visually icon in your toolbar to start editing.

## How to Use

1. Click the extension icon and toggle it **ON**.
2. Click any element on the page to select it.
3. Edit styles using the sidebar — fonts, colors, spacing, layout, shadows.
4. Click **"✨ Generate Copilot Prompt"** to copy a ready-to-use `gh copilot suggest` command.
5. Paste it in your terminal and let Copilot do the rest.

## Built With

- Vanilla JavaScript
- Chrome Manifest V3
- Zero dependencies
